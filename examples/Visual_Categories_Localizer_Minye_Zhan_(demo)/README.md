1. If not yet one, install Gostim2 from <http://github.com/chrplr/gostim2>

2 run:

   cd examples/Visual_Categories*
   gostim2 -csv visual_categories_minye_zhan.csv -assets stimuli

Or use `gostim2-gui`

Reference:

> Zhan, M., Pallier, C., Agrawal, A., Dehaene, S., & Cohen, L. (2023). Does the visual word form area split in bilingual readers? A millimeter-scale 7-T fMRI study. Science Advances, 9(14), eadf6140.


To request full stimuli sets and timings, contact Minye Zhan at <zhanminye@gmail.com>
